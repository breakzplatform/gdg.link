gdg.link

gdg.link is a free url shortner and subdomain service for Google Developer Groups community. Use it for your own GDG pages, events and for share links with your community.

gdg.link can be used on tree ways:

¹go.gdg.link

It's the most basic use, generating a random short url for a specifc link. You can short a link by using this form below. If you need an API key for something you are developing on, ask me on Twitter.

²gdg.link/myurl

If you want a custom url and use directly the gdg.link domain, fork the repo and create a file myurl.html (where myurl is the url that will be used) on the gh-pages branch. Then, redirect it to your page by setting the window.location.href via JavaScript (how?). After that, make a Pull Request on GitHub. The urls are distributed as "first come, first served".

³myurl.gdg.link

If you want a sub-domain and control it on with your host, it's possible too. You can use it on GitHub Pages, Firebase Hosting and pretty much any place where we can point a CNAME. For have a subdomain for your own, create a issue on gdg.link repo, say what is the subdomain you want, why you want it, where the DNS shout point and tag it with "DNS". Wait a couple time to configure your DNS servers and you are good to go. "Why?" is important because we have, at least for now, a limit of 3000 subdomains. The subdomains are also distributed as "first come, first served".
